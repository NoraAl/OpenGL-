# OpenGL Sample
Simple sample for Mac users.

## Build and Run
After downloading, within the root directory:
```sh
cd build
cmake ..
```
Once the automatic Makefile is generated, you can run:
```
make 
./shapes
```
If you are using **CLion**, you can open the project directly.
